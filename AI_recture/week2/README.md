# WEEK2 アヤメの分類
今週はアヤメの分類を考え、[sigate](https://signate.jp/competitions/115/data)で提出してもらいます。
ただし、NNを用いて実装して下さい。
<br>時間が余った人は[称号限定コンペ](https://signate.jp/competitions/750)で、ボーダーを超えて下さい。

手順
１．アカウントを作る。
２．データをローカルPCにダウンロードする。
３．提出形式を確認する。
４．データを可視化してみる
５．モデルを構築
６．提出

意図：
・分類タスクに取り組む。
・データの入出力を確認する。
・データを可視化する。
